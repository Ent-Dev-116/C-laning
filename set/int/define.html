define

defineの入力の仕方

defineは#define ① ②のような入力法になります。
また、この文章は#includeと同じで、外に書かなければいけません。

①

①には変数名のようなものが入ります。ここは、一般的に大文字で書くようになっています。

②

②には変数の中に入る値が入ります。
しかし、気を付けなければならないのが、変数のように後で書き換えることはできないということです。
つまり、今後は「①」を「②」という値に置き換えますという意味になります。

今までの説明を分かりやすくするサンプルコード

#include<stdio.h>

/**********************************************
    define 説明用サンプルコード            
    作成者:換気扇  
    推奨ファイル名: define_test.c          
**********************************************/

//defineを使用する。これは九九の大きさの指定 最大値は999
#define LENG 9

int main(void){
        //変数宣言
        int i,j;

        //何の出力かわかるような装飾
        printf("---|--掛け算の表----");
        for(i=0;i<LENG-4;i++){
            printf("----");
        }
        printf("\n   |\n");

        //九九の票を出力
        for(i=1;i<=LENG;i++){
            printf("%3d|",i);
            for(j=1;j<=LENG;j++){
                printf("%4d",i*j);
            }
            printf("\n");
        }
        printf("   |\n");


        return 0;
}

サンプルコード徹底解説

ここでは、サンプルコードで果たして何をしていたのかを表します。

1,#include ... (void){～定型文～

ザ・定型文です。
今のうちは、「始めの合図」だと思ってください。

2,#define LENG 9～defineの確認～

defineの使い方の確認です。これにより、今後の「LENG」は「9」になります。
こうすることで、今後「9」を「99」にするだけで「九九の表」を「九九九九の表」にすることができます。

3,int i,j; ... printf(" |\n");～考えてみよう～

説明を記述するのもいいかな…と思いましたが、ほぼ全て今まで解説してきた中級編の物なので、何が起きてるかを考えてみましょう。
面倒になったわけではありません。決して。

4,retu ... }～定型文～

「1,」の内容が始めの合図だとすると、こっちは「終わりの合図」です。
「1,」と「4,」が何なのかは追々分かります。
