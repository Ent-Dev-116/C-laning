for

forの入力の仕方

forはfor(（　①　）;（　②　）;（　③　）){（　④　）};のような入力法になります。

①

最初に実行される関数をここに記述します。
よく使うのは[i=0]で、こうすると最初にこの後使う変数を初期化できます。
それ以外にも「最初に実行される関数」なので、色々悪さできますが…最後のおまけに書きます。

②

継続条件をここに記述します。
whileでも言ったように、継続条件なので、その条件が満たされている間続けます。
間違えないように。

③

終了処理をここに記述します。
終了処理というのは、最後に終わった時に実行される関数の事です。

④

処理をここに記述します。
whileの{}の中身と同じです。基本的に中括弧の中身は条件が満たされている時実行される処理を書く認識で大丈夫です。

今までの説明を分かりやすくするサンプルコード

#include<stdio.h>

/**********************************************
    for 説明用サンプルコード            
    作成者:換気扇  
    推奨ファイル名: for_sanp.c        
**********************************************/

int main(void){

        int i,rop;

        printf("繰り返す回数を入力してください。\n->");
        scanf("%d",&rop);

        //ループの処理確認（表示回数0からはわざと）
        for(i=0;i<rop;i++){
            printf("\nこれは、%d回表示されます（現在[%d]回目）",rop,i);
        }

        printf("\n");

        printf("\n\n---whileで再現---\n");
        //whileで無理やりforを作ろう
        //初期化処理
        i=0;
        //ループ処理,継続条件
        while(i<rop){
            printf("\nこれは、%d回表示されます（現在[%d]回目）",rop,i);

            //終了処理
            i++;
        }

        printf("\n");

        return 0;
}

サンプルコード徹底解説

ここでは、サンプルコードで果たして何をしていたのかを表します。

1,#include ... (void){～定型文～

ザ・定型文です。
今のうちは、「始めの合図」だと思ってください。

2,int rop,i; ... "%d",&rop);～入力処理～

いつもの変数宣言と入力。変数とその役割は下記の通り。

  • [i]...for文の回数指定ループによく使うアルファベット一文字。今回もそれに使用。
  • [rop]...ループ回数の指定数を格納する変数。roopの略。
3,for(i=0;i<rop;i++){ ... }～for文の確認～

ループの動作を確認しています。
初期化処理は[i=0]でカウントを0にし、
条件は[i<rop]で指定したループ回数より小さい回数繰り返し、
終了処理で[i++]をすることでカウントを1ずつ上昇させます。

4,i=0; ... }～whileでfor再現～

whileで無理やりfor文を再現した形です。
ほぼ確実に使いませんが、一応whileのこの処理がforで短縮できるという説明のために記述しました。

5,retu ... }～定型文～

「1,」の内容が始めの合図だとすると、こっちは「終わりの合図」です。
「1,」と「5,」が何なのかは追々分かります。

オマケ[forでよくあるエラー]

セミコロン抜け

whileとは違い、()の中にはセミコロンが二つ必要になるので、これを忘れてエラーになりがちです。
