四則演算（一般）

四則演算の入力の仕方

四則演算は、記号で入力します。掛け算は*、割り算は/のような入力法になります。
足し算引き算は、+,-で変わりません。
余りを求めるときには、%を使用します。

今までの説明を分かりやすくするサンプルコード

#include<stdio.h>

/**********************************************
    四則演算 説明用サンプルコード            
    作成者:換気扇 
    推奨ファイル名:cul_sanp.c          
**********************************************/

int main(void){

        //変数宣言
        int num1,num2,ans;
        
        //計算するための値を入力
        printf("num1\n>");
        scanf("%d>",&num1);
        printf("num2(0以外)\n>");
        scanf("%d>",&num2);

        //計算
        ans=num1+num2;
        printf("\n演算子(+)を使用した足し算(num1+num2)\n%d\n",ans);

        ans=num1-num2;
        printf("\n演算子(-)を使用した引き算(num1-num2)\n%d\n",ans);
     
        ans=num1*num2;
        printf("\n演算子(*)を使用した掛け算(num1*num2)\n%d\n",ans);
        
        ans=num1/num2;
        printf("\n演算子(/)を使用した割り算(num1/num2)\n%d\n",ans);

        ans=num1%num2;
        printf("\n演算子(%)を使用した余りを求める演算(num1%num2)\n%d\n",ans);

        //最後の改行
        printf("\n");

        return 0;
}

サンプルコード徹底解説

ここでは、サンプルコードで果たして何をしていたのかを表します。

1,#include ... (void){～定型文～

ザ・定型文です。
今のうちは、「始めの合図」だと思ってください。

2,int num1,num2,ans;～変数宣言～

前回の変数宣言そのままです。
変数の解説だけ少し入れます。

  • [num1] ... 演算をするための数字1です。[number1]の略です。
  • [num2] ... 演算をするための数字2です。[number2]の略です。
  • [ans] ... 答えを保存するための変数です。[answer]の略です。
3,printf("num1\n..."%d>",&num2);～変数に入力～

変数に代入します。これで自由な値を入れて、四則演算を自由な値で行えます。

4,printf("\n演算子(+)...割り算(num1/num2)\n%d\n",ans);～演算～

四則演算が使用できることの確認です。

流れは、

  • scanfで変数の中に入れる
  • ↓
  • 四則演算を行った値を[ans]に入れる
  • ↓
  • printfで[ans]に入った物を表示する
  • ↓
  • 上二つを色々な演算で試す

という順番です。

5,retu ... }～定型文～

「1,」の内容が始めの合図だとすると、こっちは「終わりの合図」です。
「1,」と「5,」が何なのかは追々分かります。


四則演算（短縮記法）

短縮記法とは？

短縮記法は、いままで書いていた演算をより分かりやすく書くための技術です。
使わなくてもできますが、使うともっとわかりやすくなります。

num1=num1+num2;->num1+=num2;

「ある変数1」と「ある変数2」を計算した後に、「ある変数1」に代入するときに使います。
+=だけでなく、*=などの上記の四則演算ならすべて使えます。

num1=num1+1;->num1+=1;->num1++;

上記のように短縮記法を短縮するという事ができます。
「ある変数」に単純に「1」を加算したいときに使えます。そして、num1--;のように、単純に「1」を引きたいときにも使えます。
気を付けるべきは、num1+=1;の省略記法なので、「ある変数」に加算された値が勝手に代入されます。
++num1;という、他の計算より先に「1」を加算する方法もありますが、実用的ではないのであまり用途はありません。


今までの説明を分かりやすくするサンプルコード

#include<stdio.h>
    
/**********************************************
    四則演算 説明用サンプルコード            
    作成者:換気扇 
    推奨ファイル名:cul_short_sanp.c          
**********************************************/
    
int main(void){
    
        //変数宣言
        int num1,num2;
            
        //計算するための値を入力
        printf("num1\n>");
        scanf("%d>",&num1);
        printf("num2(0以外)\n>");
        scanf("%d>",&num2);
    
        //計算
        num1*=num2;
        printf("\n演算子(*=)を使用した掛け算(num1*=num2)\n%d\n",num1);
    
        num2++;
        printf("\n演算子(++)を使用した加算(num2++)\n%d\n",num2);

        //最後の改行
        printf("\n");

        return 0;
}

サンプルコード徹底解説

ここでは、サンプルコードで果たして何をしていたのかを表します。

1,#include ... (void){～定型文～

ザ・定型文です。
今のうちは、「始めの合図」だと思ってください。

2,int num1,num2;～変数宣言～

前回の変数宣言そのままです。
変数の解説だけ少し入れます。

  • [num1] ... 演算をするための数字1です。[number1]の略です。こっちのサンプルコードでは[num2]との掛け算を代入します。
  • [num2] ... 演算をするための数字2です。[number2]の略です。こっちのサンプルコードでは[num2]を加算した値を代入します。

省略記法の演算が使用できることの確認です。
流れは前のコードと同じです。

オマケ[四則演算と省略記法でよくあるエラー]

変数型の違い

恐らく整数同士の割り算を見て分かったと思いますが、小数点以下切り捨ての為、[1/3]は[0]になります。
そのため、[float]を使う必要があるのですが、整数と小数の型が違うと計算できません。計算の時は、変数や数値の型をそろえるよう注意しましょう。
