数値型変数（int,float,long）

大まかな変数分類

変数はいくつか型があります。
型というのはいわば数値の形のことです。

覚えなくてはならないのが、「整数値のint型」「小数値のfloat型」「一文字のchar型」です。
あとは下の表へ。

入力 何が代入できるか 変数の範囲
int 整数 -2147483648 ～ 2147483647
float 小数表示 3.4E-38 ～ 3.4E+38
char アルファベット一文字 -128 ～ 127
short int 小さい整数。容量が軽い -32768 ～ 32767
long int 大きい整数。容量が重い -2147483648 ～ 2147483647
double 大きい小数。 1.7E-308 ～ 1.7E+308
unsigned int 負数を含まない整数。長くなる。 0 ～ 4294967295

変数にくっつく便利な奴ら

long

長くしてくれる。主に、intの時に使う。ただし、「%d」等と書くときに「%ld」と入れなくてはならない。

unsigned

無駄な値を無くしてくれる。int等の数字型であれば、負数をなくして、その分長くする。
long unsigned intと宣言すれば、正の数では取りえる範囲が最長になる。

変数宣言の方法

[変数型名] [変数名];

今までの説明を分かりやすくするサンプルコード

#include<stdio.h>

/**********************************************
    変数 説明用サンプルコード

int main(void){
        //変数宣言
        int a=1;

        //代入している値は[long int]の最大値
        long int a=9223372036854775807;
        
        float b=0.1;
        return 0;
}

サンプルコード徹底解説

ここでは、サンプルコードで果たして何をしていたのかを表します。

1,#include ... (void){～定型文～

ザ・定型文です。
今のうちは、「始めの合図」だと思ってください。

2,int a; ... float b=0.1;～変数宣言～

変数宣言です。今まで[printf]などで見せてきましたので、何となく理解していると思います。
少し詳しく説明すると、この変数宣言をすることで、数字を補完するいわば「箱」を作っているイメージです。

3,retu ... }～定型文～

「1,」の内容が始めの合図だとすると、こっちは「終わりの合図」です。
「1,」と「3,」が何なのかは追々分かります。

オマケ[()でよくあるエラー三選]

1.変数名が被る

時々あるやつです。名前は"a"とか"b"じゃなくてちゃんと意味を込めましょう。

後で見たときとかも楽になるよ。

2.予約語を使う

あまりないです。というか変数宣言でミスはしません。ほぼ。

3.変数型と違うものを使う

「変数型を勘違いしていて、[float]に文字を入れてしまった」とかです。

お察しの通り、ミスはないです。がちで。

オマケ[最大値が知りたい]

最大値を知る方法

一応表に書きました。が、わかりづらく無いですか？
どうせなら実行すれば、勝手に教えてくれるようにしたいですよね？
というわけで、下のコードを作りました。

#include<stdio.h>
#include<limits.h>
#include<float.h>
    
int main(){

        printf("int型\n%d ~ %d\n\n",INT_MIN,INT_MAX);
        printf("char型\n%d ~ %d\n\n",CHAR_MIN,CHAR_MAX);
        printf("float型\n%e ~ %e\n\n",-FLT_MAX,FLT_MAX);
        printf("double型\n%e ~ %e\n\n",-DBL_MAX,DBL_MAX);

        printf("long int型\n%ld ~ %ld\n\n",LONG_MIN,LONG_MAX);
        printf("unsigned int型\n0 ~ %u\n\n",UINT_MAX);
        printf("short int型\n%d ~ %d\n\n",SHRT_MIN,SHRT_MAX);
        
        return 0;
}

このコードを実行すれば、いつでも最大最小値を知れます。
